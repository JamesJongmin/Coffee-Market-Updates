적용된 스타일 변경사항

/* 기본 메뉴 아이템 */
.menu-item {
    word-wrap: break-word;
    line-height: 1.4;
}

/* In-depth & Fundamentals 최적화 */
#analysisMenu, #fundamentalsMenu {
    font-size: 1.05em;
    padding: 20px 30px;
}

/* 태블릿 (768px) */
@media (max-width: 768px) {
    .menu-item {
        font-size: 1em;
        padding: 16px 25px;
    }
    
    #analysisMenu, #fundamentalsMenu {
        font-size: 0.95em;
        padding: 18px 25px;
        line-height: 1.5;
    }
}

/* 모바일 (480px) */
@media (max-width: 480px) {
    .menu-item {
        font-size: 0.9em;
        padding: 14px 20px;
    }
    
    #analysisMenu, #fundamentalsMenu {
        font-size: 0.85em;
        padding: 16px 20px;
        line-height: 1.6;
        white-space: normal;
    }

최적화 특징

  • 자동 줄바꿈: word-wrap: break-word로 긴 텍스트 자동 줄바꿈
  • 행간 조정: line-height를 1.4~1.6으로 설정하여 가독성 향상
  • 반응형 글씨 크기:
    • 데스크톱: 1.05em
    • 태블릿: 0.95em
    • 모바일: 0.85em
  • 패딩 최적화: 화면 크기에 따라 적절한 여백 조정
  • white-space: normal: 모바일에서 필요시 텍스트 줄바꿈 허용
