⚡ 주요 기능들

🔍 자동 추적되는 데이터:

  • 유입 소스: Google, Naver, Facebook, Twitter 등 자동 감지
  • UTM 파라미터: 캠페인 추적 (?utm_source=google&utm_medium=cpc)
  • 클릭 이벤트: 사용자가 클릭하는 모든 요소 추적
  • 디바이스 정보: 모바일/데스크톱/태블릿 자동 분류
  • 실시간 접속자: 5분 이내 활성 사용자 실시간 카운트

📊 분석 가능한 데이터:

  • 어디서 클릭해서 들어왔는지: 검색엔진, 소셜미디어, 직접방문 구분
  • 시간대별 방문 패턴: 언제 가장 많이 방문하는지 분석
  • 사용자 행동 분석: 클릭 위치, 체류 시간, 페이지 이동 패턴
  • 디바이스별 접속 비율: 모바일 vs 데스크톱 사용 패턴
  • 실시간 활성 사용자: 현재 사이트를 보고 있는 사용자 수
🧪 테스트 방법
💡 팁: 실제 환경에서는 아래와 같은 URL로 접속하면 다양한 유입 소스를 시뮬레이션할 수 있습니다!
# Google 검색에서 온 것처럼 시뮬레이션 http://yoursite.com?utm_source=google&utm_medium=organic # Facebook에서 온 것처럼 시뮬레이션 http://yoursite.com?utm_source=facebook&utm_medium=social # 캠페인 추적 테스트 http://yoursite.com?utm_source=newsletter&utm_medium=email&utm_campaign=weekly

🔧 개발자 도구에서 테스트:

  • window.trafficAnalytics.generateTrafficStats() - 현재 통계 확인
  • toggleTrafficDashboard() - 실시간 대시보드 열기
  • localStorage.getItem('trafficLogs') - 저장된 데이터 확인
